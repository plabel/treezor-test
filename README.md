# treezor-test

Application Laravel web sécurisée respectant le modèle MVC. On affiche les écrans CRUD de l’entité utilisateur.

## Installation

Cloner ce repo et installer les dépendances avec `composer install`.

## Usage

Lancer docker-compose up.
